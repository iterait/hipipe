9 __HiPipe__ is a C++ library for efficient data processing. Its main purpose is to simplify
10 and acclelerate data preparation for deep learning models, but it is generic enough to be used
11 in many other areas.
12 
13 __HiPipe__ lets the programmer build intuitive data streams that transform,
14 combine and filter the data that pass through. Those streams are compiled,
15 batched, and asynchronous, therefore maximizing the utilization of the provided
16 hardware.
17 
18 - [Documentation and API reference](https://hipipe.org/).
19 - [Installation guide](https://hipipe.org/installation.html).

21 ## Example
22 
23 ```c++
24 std::vector<std::string> logins = {"marry", "ted", "anna", "josh"};
25 std::vector<int> ages = { 24, 41, 16, 59};
26 
27 auto stream = ranges::view::zip(logins, ages)
28 
29  // create a batched stream out of the raw data
30  | cxs::create<login, age>(2)
31 
32  // make everyone older by one year
33  | cxs::transform(from<age>, to<age>, [](int a) { return a + 1; })
34 
35  // increase each letter in the logins by one (i.e., a->b, e->f ...)
36  | cxs::transform(from<login>, to<login>, [](char c) { return c + 1; }, dim<2>)
37 
38  // increase the ages by the length of the login
39  | cxs::transform(from<login, age>, to<age>, [](std::string l, int a) {
40  return a + l.length();
41  })
42 
43  // probabilistically rename 50% of the people to "buzz"
44  | cxs::transform(from<login>, to<login>, 0.5, [](std::string) -> std::string {
45  return "buzz";
46  })
47 
48  // drop the login column from the stream
49  | cxs::drop<login>
50 
51  // introduce the login column back to the stream
52  | cxs::transform(from<age>, to<login>, [](int a) {
53  return "person_" + std::to_string(a) + "_years_old";
54  })
55 
56  // filter only people older than 30 years
57  | cxs::filter(from<login, age>, by<age>, [](int a) { return a > 30; })
58 
59  // asynchronously buffer the stream during iteration
60  | cxs::buffer(2);
61 
62 // extract the ages from the stream to std::vector
63 ages = cxs::unpack(stream, from<age>);
64 assert((ages == std::vector<int>{45, 64}));
65 ```
